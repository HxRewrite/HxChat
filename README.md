# HxChat

HxChat is a Minecraft plugin for Paper/Folia servers that provides enhanced chat functionality with features to improve player communication.

## Features

- Enhanced chat formatting and functionality
- Permission-based system for feature access
- Compatible with both Paper and Folia server implementations
- Designed for Minecraft 1.21.8+

## Installation

1. Download the plugin JAR file
2. Place the JAR file in your server's `plugins` folder
3. Restart or reload your server
4. The plugin is now ready to use!

## Permissions

- `hxchat.use` - Allows using the chat commands (default: true for all players)
- `hxchat.admin` - Allows using admin chat features (default: op)
- `hxchat.*` - All chat permissions (default: op)

## Usage

The plugin provides enhanced chat functionality for your Minecraft server.

## Requirements

- Java 21 or higher
- Paper or Folia 1.21.8+ server
- Minecraft 1.21.8+
